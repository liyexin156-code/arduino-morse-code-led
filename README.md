# Arduino Morse Code LED

## Description
This project converts user input text into Morse code using an LED on Arduino.

## Features
- Input text via Serial Monitor
- LED outputs Morse code (dot and dash)
- Supports A-Z and spaces

## Hardware
- Arduino Uno
- LED
- Resistor
- Breadboard

## How to Use
1. Upload the code to Arduino
2. Open Serial Monitor (9600 baud)
3. Enter text (e.g., HELLO)
4. Observe LED blinking Morse code

## Concepts Used
- Arrays (character to Morse mapping)
- Serial communication
- Timing control
